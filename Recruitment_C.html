香港科技大学（广州）简介

香港科技大学（简称港科大）自1991年成立以来，凭借其锐意创新、与时并进的理念，在全球高校中迅速崛起，已经发展为一所在国际上享有盛誉的研究型大学。 香港科技大学（广州）（简称港科大（广州）），作为一所具有独立法人资格的内地与香港合作办学机构，位于广东省广州市南沙区，成立于2022年，与港科大共同构成“科大2.0”。 在“科大2.0”核心战略的背景下，港科大（广州）采用了全新的、融合学科的学术架构。 该架构以“枢纽”（Hub）和“学域”（Thrust）取代传统学科的“学院”和“学系”模式，旨在推动学科交叉融合，与香港科技大学清水湾校区的深厚学科根基相辅相成，实现资源共享、优势互补，以应对世界和人类社会面临的复杂挑战。

在学生培养上，港科大（广州）采用全英文教学，旨在培养具有国际视野和创新能力的高端人才，以服务于粤港澳大湾区的科技创新和产业升级。 在“港科大一体，双校互补”的指引下，两校紧密合作，教学和研究设施相互开放，课程学分互认，并确保两校学位证书、学生待遇的一致性。 在香港科技大学（广州）就读的硕士、博士研究生将获得由香港科技大学颁发的相应学位证书。

博士生和研究助理（RA）的招聘要求和待遇

实验室研究涉及人工智能，形式化方法，控制理论，机器人学，以及通信科学等领域的相互交叉，因此诚邀具有形式化方法与自动机理论，机器学习，自动化与控制理论，运筹学和博弈论等方面背景的同学加入实验室。以下是对申请者的具体要求：

  1. 拥有（但不限于）计算机科学，自动化，数学与应用数学，力学，电气工程等专业本科或硕士学位（即将毕业的同学也可以提前联系）；

  2. 学习优秀，热爱科研，有团队合作精神，有志于在安全可信人工智能领域做出具有原创性的理论和实践成果；

  3. 有良好的数学基础， 熟练的编程能力， 和较强的英语写作与沟通能力，满足香港科技大学（广州）的英语语言入学要求；

  4. 对符合以下条件的申请者优先考虑：

    1. 有一定科研经历，有人工智能，形式化方法，控制等领域期刊和会议论文发表经验；

    2. 有与无人机，无人船等无人系统相关的研究或实习经验。

博士研究生的学制为 3 年（对于已有相关研究型硕士学位的申请者）或 4 年（对于没有相关研究型硕士学位的申请者），学费为4万人民币/年，所有录取的博士生将获得全额奖学金（约1.5万人民币/月）。 对本实验室感兴趣但暂时缺乏经验的同学可以考虑申请研究助理（RA）。 实验室将提供有竞争力的津贴，并优先考虑表现优异者的博士申请。

博士后的招聘要求和待遇

  1. 已取得（3年内）或即将取得国内外大学或研究机构计算机科学，自动化，人工智能等相关专业的博士学位。

  2. 具有扎实的研究功底和自主探索的能力，博士期间研究与实验室研究领域相关。

  3. 博士后聘期为2年，提供40万-60万综合年薪，各级政府提供最高45万元科研经费。

  4. 支持博士后在站期间申请各级科研课题资助，协助申请市/区级各项人才称号及人才待遇。

  5. 出站后留粤工作的博士后可申请最高70万元安家费。

  6. 博士后人员进站，可选择落户广州市，其配偶及未成年子女随迁入户、子女入托、入学等按广州市相关条例执行。

硕士研究生招聘要求

硕士研究生申请人无需提前联系导师，直接通过学校官网申请即可。如已经被硕士项目录取并希望加入实验室开展科研工作，请直接发送邮件联系。

申请方式

请将个人简历（中英文皆可）、所有学位的成绩单、代表性论文或项目（如有）以及研究概述（英语）发送到：bingzhuoz(at)hkust-gz(dot)edu(dot)cn，邮件主题请按照“PhD/RA/PostDoc Application + Name”格式填写。 在研究概述中，请简要介绍申请人的研究背景及其对后续研究的构想。 如果申请通过筛选，将尽快通过电子邮件通知面试的相关事宜。
